For more privacy, you can store your login/password with YOUR rsa key.

0) Exit program

1) Run the following .bat/.sh script (inspired by http://stackoverflow.com/questions/3441501/java-asymmetric-encryption-preferred-way-to-store-public-private-keys)
set RANDFILE=.rnd
openssl genrsa -out vkey.pem 2048
openssl pkcs8 -topk8 -nocrypt -in vkey.pem -inform PEM -out vkey.private.der -outform DER
openssl rsa -in vkey.pem -pubout -outform DER -out vkey.public.der
rm .rnd
rm vkey.pem

2) In your home folder (%userprofile% in Windows, ~ in Linux) open `.vk-msg-notify` file with text editor.
Replace it the following content (set YOUR path to files; pay attention to double backslashes):
{
    "RSAKeysFileNames" : {
        "public" : "path\\to\\your\folder\\vkey.public.der",
        "private" : "path\\to\\your\folder\\vkey.private.der"
    }
}

3) Run program again
